사용자가 처음으로 웹 페이지를 배포하기 위한 아주 간단한 단일 HTML 파일을 만들어줘.

기술 스택: HTML, CSS (Tailwind CSS를 CDN 방식으로 사용해서 별도의 CSS 파일 없이 만들어줘).

디자인: 영화 정보를 담는 카드 보드 형태의 게시판 레이아웃. 배경은 어두운 톤으로 하고, 카드는 깔끔한 흰색이나 밝은 회색 계열로 구성해줘.

카드 구성 요소:

영화 포스터 (이미지 주소: url)

영화 제목 (살목지, Salmokji : Whispering Water (2025))

시놉시스 (제공된 텍스트를 요약하지 말고 그대로 넣어줘)

특징: 반응형으로 제작해서 모바일과 PC에서 모두 잘 보이게 해줘.
